# Home Buying Prediction using ANN

This project predicts whether a person can afford to buy a home based on income, expenses, and savings.

## Technologies
- Python
- Pandas
- Scikit-Learn
- TensorFlow / Keras

## Model
Artificial Neural Network (ANN)

## Features
Income
Age
Dependents
Occupation
City Tier
Rent
Expenses
Savings
Disposable Income

## Output
Probability of buying a home.
